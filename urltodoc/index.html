App Info

This is a small application solely for exporting Word docx file versions of live Canada.ca content. Useful for local web content authoring and editing.

How to use:

Simply enter an English OR French www.canada.ca URL, then click Load Content. The relevant content will populate the preview area (with no header/footer) and export options will be made available. Use the preview for reference only as pages may not be rendered correctly 1:1.

IMPORTANT: This may not work for all Canada.ca subdomains (like parks.canada.ca), nor any *.gc.ca and any other external website. For best results, pages must be hosted by AEM on Canada.ca (like www.canada.ca/en/government/system.html)

Developed with help from AI (Claude and Gemini). However, it's important to note that this app does NOT actively use AI in any way. No user data is being collected or stored in any form.

Originally developed for web services needs at Treasury Board of Canada Secretariat. Other helpful web-related tools available:

  • LinkBuddy
